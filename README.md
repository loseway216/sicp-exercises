## 环境

- 安装 racket 包
- zshrc 配置 PATH
- raco pkg install sicp
- raco pkg install racket-langserver
- vscode 安装 magic racket 插件
- raco pkg install fmt, 通过 runOnSave 插件执行命令 raco fmt -i filepath
